# VPN Connection Issue Troubleshooting

## Issue

User is unable to connect to VPN.

## Possible Causes

* Incorrect credentials
* No internet connection
* VPN client issue
* Server downtime

## Troubleshooting Steps

1. Verify internet connection
2. Confirm username and password
3. Restart VPN application
4. Reinstall VPN client if needed
5. Try connecting to a different server

## Resolution

VPN connected successfully after updating credentials and restarting the client.

## Escalate If

* Multiple users report VPN failure
* VPN server is down
* Issue persists after reinstall
